# NatureNest-Bootstrap-Blog
A responsive multi‑page nature blog built with Bootstrap 5, featuring a home carousel, featured articles, an articles page, and a contact form.
NatureNest – Bootstrap Blog

 Features

- Responsive navbar with offcanvas menu
- Hero carousel on the home page
- Featured article cards
- Articles listing page
- Contact form page
- Consistent styling and layout across pages
- Pages

- index.html – Home with carousel and featured articles
- articles.html – Article list/sections
- forest-conservation.html – Full article page
- contact-us.html – Contact form
  
- Tech Stack
1. HTML5
2. CSS3
3. Bootstrap 5.3.8
